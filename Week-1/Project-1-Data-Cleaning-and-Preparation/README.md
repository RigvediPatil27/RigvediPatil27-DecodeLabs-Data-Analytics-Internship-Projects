# Project 1 – Data Cleaning and Preparation

## Overview

This project focuses on cleaning and preparing a retail sales dataset using Microsoft Excel. The objective was to identify missing values, check for duplicate records, standardize data formats, and ensure the dataset is accurate and ready for analysis.

## Objectives

- Identify missing values
- Remove duplicate records
- Standardize date formats
- Validate numerical data
- Prepare the dataset for analysis

## Tools Used

- Microsoft Excel

## Skills

- Data Cleaning
- Data Validation
- Data Preparation
- Spreadsheet Analysis
